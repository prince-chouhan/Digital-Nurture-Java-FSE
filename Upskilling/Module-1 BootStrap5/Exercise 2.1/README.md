# Bootstrap Directory Structure

## CSS Folder
Contains Bootstrap stylesheets.

Examples:
- bootstrap.css
- bootstrap.min.css

Purpose:
Provides styling for buttons, forms, grids, cards, typography, alerts, etc.

---

## JS Folder
Contains Bootstrap JavaScript files.

Examples:
- bootstrap.js
- bootstrap.bundle.js
- bootstrap.bundle.min.js

Purpose:
Provides interactive components like:
- Modal
- Dropdown
- Accordion
- Collapse
- Tooltip

---

## Icons Folder
Contains Bootstrap Icons.

Purpose:
Provides SVG-based icons that can be used in webpages.

Examples:
- House Icon
- Person Icon
- Search Icon
- Facebook Icon
